Overview

Vidiun API documentation provides information to web developers who wish to use Vidiun's Application Programming Interfaces (APIs). To gain a wider understanding of the Vidiun open source video platform, please review the Introduction to Vidiun Platform document.

The Vidiun open source video platform exposes its Server API to web applications by implementing a standard HTTP POST/GET url-encoded requests structure. Vidiun server API's, also named Vidiun Partner Services, are designed to follow REST principles. The server APIs consist of service actions for querying, setting, updating and listing entities as well as for activating processes within Vidiun open source video platform. Service actions are grouped according to the entity type they are applied on, and provide all actions relevant to the specific entity.

This API documentation provides specific information on:

  • Vidiun Services and their related actions
  • Vidiun Objects and their related properties
  • Vidiun Enumerated Types and their values

A prerequisite for using Vidiun APIs is obtaining Vidiun partner identifiers, such as Partner ID. Learn more about obtaining Vidiun partner identifiers.

In addition to this documentation, Vidiun provides an API Test Console Application, allowing you to develop applications using the Vidiun API.

For additional support, please visit Vidiun Community Forums.
